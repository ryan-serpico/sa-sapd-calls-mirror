## Getting Started
If you want to run this code locally, here you go.

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/ryan-serpico/sa-sapd-calls-mirror.git
   ```
2. Install packages
   ```sh
   pip3 install -r requirements.txt
   ```
3. Run the bot
   ```sh
   python3 app.py
   ```
